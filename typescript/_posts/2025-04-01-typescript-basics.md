## Basic types

### boolean

```TypeScript
let x: boolean = true;
```

### number

```TypeScript
let x: number = 1;
```

### string

```TypeScript
let x: string = 'string';
```

### string[]

Arrays of strings, similar to lists in Python

```TypeScript
let x: string[] = ['a', 'b', 'c'];
console.log(x[0]);
```
